Разработал и реализовал высококонверсионную посадочную страницу-витрину, объединяющую несколько товарных категорий в едином интерфейсе. 
Цель проекта — оптимизация рекламного бюджета за счёт концентрации трафика на одной странице с перекрёстными рекомендациями. 
В качестве демонстрации функционала товары были заменены на каталог книг.
